## 🎯 **ABOUT THE PROJECT**

<div align="center">
  <img src="https://media.giphy.com/media/l0HlNaQ6gWfllcjDO/giphy.gif" width="300">
</div>

**Tic Tac Toe: The Ultimate Battle Arena** is not your grandmother's tic tac toe! This isn't just a game; it's an **experience**. We've taken the classic game you know and love and supercharged it with modern design, smooth animations, and an interface that feels alive.

> *"The most beautiful tic tac toe I've ever played!" - Satisfied Player*

### 🎯 **Core Highlights**
- ✅ **Two-player gameplay** with smooth turn management
- ✅ **Real-time winner detection** - instant victory celebration
- ✅ **Draw detection** - no more confusion
- ✅ **Beautiful responsive design** - works on all devices
- ✅ **Glass morphism UI** with modern aesthetics
- ✅ **Smooth animations** and hover effects

## 🎮 **HOW TO PLAY**

<div align="center">
  
### **📖 Game Rules**

| Rule | Description |
|:---:|:---|
| **1️⃣** | Player **X** always goes first |
| **2️⃣** | Players alternate placing marks in empty squares |
| **3️⃣** | Get **3 in a row** (horizontally, vertically, diagonally) to win |
| **4️⃣** | If all 9 squares are filled with no winner, game is a **draw** |

### **🎯 Winning Patterns**

```
🏆 HORIZONTAL WINS        🏆 VERTICAL WINS         🏆 DIAGONAL WINS

[X] [X] [X]    [ ] [ ] [ ]    [X] [ ] [ ]    [ ] [X] [ ]    [X] [ ] [ ]
[ ] [ ] [ ]    [X] [X] [X]    [X] [ ] [ ]    [ ] [X] [ ]    [ ] [X] [ ]
[ ] [ ] [ ]    [ ] [ ] [ ]    [X] [ ] [ ]    [ ] [X] [ ]    [ ] [ ] [X]

 Row 1 Win       Row 2 Win      Col 1 Win      Col 2 Win      Diag Win
```

### **🎮 Controls**

| Control | Action |
|:---:|:---|
| 🖱️ **Click Square** | Place your mark (X or O) |
| 🔄 **Reset Button** | Clear board and start over |
| 🆕 **New Game Button** | Same as reset |

## 🧠 **GAME LOGIC EXPLAINED**

<div align="center">
  
### **🎯 Winning Patterns Array**

```javascript
const winPatterns = [
    [0,1,2], // Top row
    [3,4,5], // Middle row
    [6,7,8], // Bottom row
    [0,4,8], // Diagonal
    [2,4,6], // Anti-diagonal
    [0,3,6], // Left column
    [1,4,7], // Middle column
    [2,5,8]  // Right column
];
```

### **⚙️ Winner Detection Algorithm**

```javascript
const checkWinner = () => {
    for(let pattern of winPatterns){
        let a = boxes[pattern[0]].innerText;
        let b = boxes[pattern[1]].innerText;
        let c = boxes[pattern[2]].innerText;
        
        if(a && b && c && a === b && b === c){
            showWinner(a);
            return; // Exit immediately!
        }
    }
    if(checkClick === 9) showDraw();
};
```
